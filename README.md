# vue-forum

vue-forum is a project that created using Vue.js and Firebase, during vueschool MasterClass Course

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
